### 介绍

技能冷却计时器 不读取、修改游戏内存，<br/>
仅通过监听键盘事件帮助你计算技能冷却时间和技能效果持续时间。

你需要使用窗口化运行游戏或在全屏游戏中点击一次公告 ( 玄学操作 ) 才能看到悬浮窗。


### 运行环境要求

- Windows 10 x64 及以上的非精简版系统
- Windows 管理员权限
- 机灵的小脑瓜
